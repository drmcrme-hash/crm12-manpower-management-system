# CRM 1&2 Electrical Manpower Management System

## Version
V3.0 Enterprise (Under Development)

## Organization
Cold Rolling Mills (CRM 1 & 2)
Bokaro Steel Plant
Steel Authority of India Limited (SAIL)

## Description

CRM 1&2 Electrical Manpower Management System is a web-based application developed for managing manpower records in the Electrical Department.

### Features

- Employee Master
- Employee Profile
- Medical Management
- Training Management
- Dashboard KPIs
- Retirement Tracking
- Excel Import/Export
- Firebase Cloud Synchronization
- Mobile Viewer
- Print Employee Profile

## Technology

- HTML5
- CSS3
- JavaScript
- Firebase Firestore
- GitHub
